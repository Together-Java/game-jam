## Game Description
1. Hunt Duke; a maze game where you have to search a maze for Duke
2. Duke's Arena: 1v1 player vs Duke game
3. Story mode "Duke's Story", cut scene/animated story providing lore on Duke

## Setup
1. Ensure Java 25 is installed
2. Run `java -jar game.jar`

## Controls

* WASD movement
* Arrow keys navigation
* T to enable flashlight
* F to interact with objects
