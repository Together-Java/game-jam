## Game Description
Minesweeper/Sokoban cross-over game

## Setup
1. Ensure Java 21 is installed
2. Run `cd SlavaDukerani`
3. Run `./mvnw compile exec:java`

## Controls
* WASD or arrow keys to move
* Q to reveal a tile (left click)
* E to flag a tile (right click)
* Mouse click to interact directly
