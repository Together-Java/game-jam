## Game Description
Reaction based jumping game.

## Setup
1. Ensure Java 21 is installed
2. Run `cd Duke-Jump`
3. Run `git clone https://github.com/TerminalVelocityCabbage/TerminalVelocityEngine2.git`
4. Run `./gradlew GameClient:run`

## Controls
* Space / W / Up arrow to jump
* Left Shift / S / Down arrow to stomp
* Escape to pause
