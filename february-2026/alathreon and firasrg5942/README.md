## Game Description
Guess Who? clone featuring Java Duke with random hats and colors.

## Setup
1. Ensure Java 25 is installed
2. Run `cd GuessTheDuke`
3. Run `./mvnw javafx:run`

## Controls
* Click a Duke in the grid, then click "Set As My Secret"
* Ask questions about hats/colors to eliminate Dukes (X marks eliminated ones)
* When only one Duke remains, click "Guess Now"
