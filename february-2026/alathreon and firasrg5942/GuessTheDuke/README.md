# GuessTheDuke
Guess Who? But with Java Duke and random assets.

## Run
```bash
.\mvnw.cmd javafx:run
```
Or: `run-game.bat`

## Play
1. Click a Duke in the grid, then "Set As My Secret"
2. AI picks its secret automatically
3. Ask about hats/colors to eliminate (X marks eliminated)
4. When 1 left, click "Guess Now"

Primary color = main, Secondary = accent.
