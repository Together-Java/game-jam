## Game Description
Archery game featuring Duke as the archer. Set power and height, then fire at the target.

## Setup
1. Ensure Java 17 is installed
2. Run `cd Duke-Does-Archery`
3. Run `./gradlew run`

## Controls
* A/Left arrow to decrease power
* D/Right arrow to increase power
* S/Down arrow to decrease height
* W/Up arrow to increase height
* Enter to fire
