## Game Description
ASCII-aesthetic puzzle game where Duke guides you through recovery phases inside a damaged system.

## Setup
1. Ensure Java 21 is installed
2. Run `cd DukeRecovery`
3. Run `./gradlew runStandalone`

## Controls
* WASD or arrow keys to move
* Double-tap a direction to rotate a maze node (left/up = counter-clockwise, right/down = clockwise)
* Q to quit
